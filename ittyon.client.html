ittyon.client

A client that communicates with a server to mirror its state.

connect!

(connect! socket)(connect! socket init-state)

Connect to a server via a bi-directional channel, and return a channel that promises to contain the client once the connection has been established. Used in conjuction with server/accept!.

send!

(send! client & messages)

Send one or more messages to the client. A message should be a transition with the time element omitted, i.e. [o e a v].

tick!

(tick! client)

Move the clock forward on the client.
